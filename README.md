# MySmartHome

Control your home with intelligent wifi-enabled lighting.

| Compatible devices / brand |
| :------------------------: |
|       Tuya Lighting        |

## Configuring TUYA API

### _**Ios**_

Create Tuya-Info.plist in root

    Key: TUYA_API_APP_KEY
    String: <TUYA_API_APP_KEY>
    Key: TUYA_API_SECRET_KEY
    String: <TUYA_API_SECRET_KEY>

Copy & Paste `Security Image` in project root dir as "_t_s.bmp_"

### _**Android**_

Create a local.properties

    tuya_api_app_key = <tuya_api_app_key>
    tuya_api_secret_key = <tuya_api_secret_key>

Copy & Paste `Security Image` in app/asrc/main/assets as "_t_s.bmp_"
